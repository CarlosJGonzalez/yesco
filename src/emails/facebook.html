What is Facebook's Locations Framework?
The Locations Framework within Facebook creates a parent-child relationship that allows for a more unified look for the brand. A “Locations” tab will be added to the corporate Facebook page and each location will be shown on that tab on the built-in map and Store Finder. This gives the local pages another way to be found on searches within Facebook. DAS Group will be moving all of the existing local Fully Promoted Facebook pages into the Locations Framework. For local pages with the name EmbroidMe, the name will be changed to Fully Promoted in order to be added to the Locations Framework.		  
 
   
 
  Because franchisees are sometimes too busy to post, the local pages will receive content from the national calendar to ensure a constant flow of content. These posts can be reviewed in the calendar on LocalFullyPromoted.com before they are posted so that franchisees can edit or opt out of posts. It is important to note that each location will still be able to share their own posts as access to the page itself will not change. Consumers will still be able to find location-specific information (address, phone number, hours, reviews, etc.).

What about the existing local Facebook pages?
Assuming the local Facebook page is a business page (as opposed to a personal profile), local Facebook pages will retain their followers and “Likes” as well as current access levels. In the event the page is a personal profile, the franchisee will need to contact Facebook to request the page be converted to a business page.		  
 
 
All local Facebook pages connected to the main/corporate page must have the same name (Fully Promoted). A location descriptor (City, ST) will be added to each page to set it apart. In the case of Fully Promoted, for example, a location’s Facebook page may be Fully Promoted (Doral, FL). The URLs for the local Facebook pages will also change according to the new page name.

For local pages still using the EmbroidMe name, it will be changed to Fully Promoted in order to be added to the Locations Framework.
 
 
  How will the transition work?
DAS Group will begin the process by “liking” every location’s local Facebook page as John Hopperman (our alias). Each location will then need to give DAS Group Admin access to their Facebook page by following the steps below.
  1. Click on Settings in the white bar at the top, right of the page
  2. Click on Page Roles on the left side of the page
  3. Go to the section labeled “Assign a New Page Role” and enter socialadmin@das-group.com in the box, select Admin from the dropdown menu, and click the blue button labeled "Add"
DAS Group will then work with Facebook to request the transition of the local pages to a location within the Locations Framework.
What if the location does not have a Facebook page?
Once the Framework is in place, if there is a location without a Facebook page, DAS Group will create a new local Fully Promoted Facebook business page.
